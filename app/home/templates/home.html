Features

  • Crawlers to scrape and store web sites with the following data provided for each page crawled:
    • social counts: tweets, g+1's, and all facebook counts
    • tag analysis: title tag, meta tags, anchor tags, h1 tags, and warnings about each
    • words: remove html, remove stop words, store plain text, word frequencies, and long tail keyword prospecting
    • note: all crawls happen in a background thread, so you may pursue other work while the spider does it's thing

  • Social counters to store data via API requests for tweets g+1s fshares based on user entered URLs

  • YouTube videos to scrape, parse, and store the title, views, likes, and dislikes from user provided videos

  • Functions to store data based on user written python functions

  • Easily export data as text, .csv, .xls, .pdf via the Copy, Print, CSV, Excel, and PDF buttons on any page where data is presented in a table
      note: these were only tested in the Chrome browser, and probably require the Adobe Flash plug-in to be installed for other browsers

  • Background job scheduling to regularly gather data for social counters, youtube channels, user functions, and crawlers
